# Figures

Generate figures for The Resolved Chord. Run from this folder:

```bash
python prediction_scatter.py
python roadmap_figure.py
python lotus_potential.py
python lotus_emblem.py
python bare_orbifold.py
python five_invariants.py
python spectral_cascade.py
python lepton_simplex.py
```

Outputs: `prediction_scatter.png`, `roadmap_figure.png`, `lotus_potential_figure.png`, `lotus_emblem.png`, `bare_orbifold.png`, `five_invariants.png`, `spectral_cascade.png`, `lepton_simplex.png`

Requires: `numpy`, `matplotlib`
